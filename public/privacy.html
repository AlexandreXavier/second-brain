Segundo Cerebro

Política de Privacidade

Última atualização: 20 de maio de 2026

Esta política descreve que dados pessoais o Segundo Cerebro recolhe, com que fim, onde ficam guardados e quais os teus direitos sobre eles. Aplica-se às versões web, desktop (Mac) e mobile (Android) da aplicação.

1. Quem somos

O Segundo Cerebro é uma biblioteca visual partilhada para guardar ideias de vídeo, posts, artigos e referências. A aplicação é operada pela equipa do projeto. Para qualquer questão sobre os teus dados, contacta-nos em jalexandre.xavier@gmail.com.

2. Dados que recolhemos

  • Informação da conta Google: ao entrar, recebemos do Google o teu nome, endereço de email e fotografia de perfil. Não temos acesso à tua password.
  • Conteúdos que guardas (capturas): URLs, títulos, descrições, categorias, notas e imagens que tu próprio adicionas à biblioteca.
  • Metadados técnicos: data e hora de criação de cada captura, identificador interno do utilizador que a criou.

3. Como usamos os dados

  • Para te autenticar e mostrar apenas a biblioteca à qual tens acesso.
  • Para guardar e apresentar as capturas que criaste, e — se ativares o filtro Toda a equipa — as capturas criadas por outros membros da mesma biblioteca.
  • Para gerar pré-visualizações automáticas dos URLs que colas (título, imagem, autor) através de serviços públicos de oEmbed.
  • Opcionalmente, para sugerir um título curto a partir do URL ou ficheiro carregado, recorrendo a um modelo de linguagem (ver secção 5).

Não usamos os teus dados para publicidade, perfilagem comercial ou venda a terceiros.

4. Onde os dados ficam guardados

Os dados são armazenados na infraestrutura do Google Firebase (projeto my-brain-9d788), nomeadamente:

  • Firebase Authentication — gestão da tua sessão.
  • Cloud Firestore — capturas, categorias, e perfis de utilizador.
  • Cloud Storage — imagens (capturas de ecrã, miniaturas) que envies.

O Google é processador de dados destes serviços, sujeito às respetivas políticas de privacidade do Google. Os dados podem ser tratados em servidores na União Europeia ou nos Estados Unidos, conforme a configuração do Firebase.

5. Terceiros (processadores)

  • Google / Firebase — autenticação, base de dados, armazenamento, e hosting da versão web.
  • Microlink, oEmbed do YouTube/X/Instagram — quando colas um URL, contactamos o respetivo serviço público para obter os metadados de pré-visualização. Nenhum dado pessoal teu é enviado, apenas o URL público.
  • OpenRouter (opcional) — se a sugestão automática de título estiver ativa, o URL é enviado a um modelo de linguagem alojado pela OpenRouter para gerar um título curto. Nenhum dado de conta é incluído neste pedido.

6. Retenção

As capturas que crias ficam guardadas até as apagares ou até pedires a remoção da tua conta. Se pedires a eliminação da conta, removemos os dados associados ao teu identificador num prazo razoável (geralmente 30 dias), com exceção de cópias de segurança operacionais que expiram automaticamente.

7. Os teus direitos

Tens o direito de:

  • Aceder aos dados que guardamos sobre ti.
  • Retificar dados incorretos.
  • Apagar a tua conta e os dados associados.
  • Exportar uma cópia das tuas capturas.
  • Apresentar uma reclamação à Comissão Nacional de Proteção de Dados (CNPD) ou à autoridade de proteção de dados do teu país.

Para exercer qualquer destes direitos, envia um email a jalexandre.xavier@gmail.com.

8. Segurança

Toda a comunicação entre o teu dispositivo e a infraestrutura do Firebase ocorre sobre HTTPS. O acesso aos dados é restrito aos utilizadores autenticados com Google Sign-In e às regras de segurança configuradas no Firestore e no Storage.

9. Crianças

O Segundo Cerebro não se destina a menores de 13 anos. Se acreditas que recolhemos inadvertidamente dados de uma criança, contacta-nos para que possamos eliminar a conta.

10. Alterações a esta política

Podemos atualizar esta política para refletir mudanças na aplicação ou na lei aplicável. A versão em vigor é sempre a publicada neste URL, com a data de “última atualização” no topo.
